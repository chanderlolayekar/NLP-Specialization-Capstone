# NLP Specialization Capstone Project

This repository serves as a comprehensive portfolio capstone encapsulating the core concepts from the **DeepLearning.AI Natural Language Processing Specialization**. 

It spans four distinct domains of NLP, moving from foundational statistical methods to state-of-the-art attention models.

## Project Architecture
1. **Classification & Vector Spaces (`course_1_classification.py`):** Sentiment analysis and word translations using Logistic Regression, Naïve Bayes, and Word Embeddings.
2. **Probabilistic Models (`course_2_probabilistic.py`):** Autocorrect, autocomplete, and Part-of-Speech (POS) tagging utilizing Dynamic Programming and Hidden Markov Models (HMM).
3. **Sequence Models (`course_3_sequence.py`):** Text generation and Named Entity Recognition (NER) powered by RNNs, LSTMs, GRUs, and Siamese Networks in TensorFlow.
4. **Attention Models (`course_4_attention.py`):** Machine translation, text summarization, and question-answering leveraging Encoder-Decoder architectures and Self-Attention (Transformers).
